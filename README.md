# **Oxygenated**
### _A lightweight and modern utility plugin for Spigot, Paper, and fork servers._

![Oxygenated Logo](https://i.imgur.com/mW8rDlP.png)

---

## Description
**Oxygenated** is a high-performance utility plugin designed for  
**admins, moderators, creators, and developers** who maintain a Minecraft server.

It provides a clean, modern MiniMessage-based messaging system and  
streamlines communication for plugins and server staff.

No unnecessary bloat. No lag. Just tools.

---

## Features
- Fully **MiniMessage-based** messages
- Flexible **configuration**
- Works on **Spigot, Paper, and forks (1.21.8)**
- Zero dependencies required

---

## Installation
1. Download the latest release of `Oxygenated.jar`
2. Place it into your server's `/plugins/` folder
3. Start the server
